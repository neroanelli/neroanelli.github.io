守护濒危雪豹｜THERMOS膳魔师雪豹杯荣获iF设计奖

2016-11-05

“守护濒危雪豹”膳魔师雪豹杯荣获 iF 设计大奖

今年秋冬，膳魔师推出的新款“雪豹杯”，以环保的理念和精致的设计，荣获了今年的 iF 家居风尚大奖。





iF 家居风尚大奖每年举办一次，是中国唯一针对家居用品行业设立的奖项，由德国设计奖机构iF主办。而德国iF国际设计论坛素来以"独立、严谨、可靠"的评奖理念闻名于世，每年评选iF设计奖有"产品设计界的奥斯卡"之称。


拥有百年历史的保温容器制造品牌膳魔师，今日携手世界自然基金会（WWF），推出系列公益产品，支持拯救濒危物种——雪豹的保护。WWF将通过长期监测和巡护、反盗猎、减少并阻止非法贸易、缓解人兽 冲突、保护雪豹栖息地等方式，减缓并阻止野生雪豹数量不断下降的危机状况。



膳魔师和世界自然基金会（WWF）的共同携手，让环保从呼吁转化为行动。




这次跨界合作之所以用“雪豹”作为承载，是因为它们的生存现状实在令人堪忧。

广受人们喜爱的雪豹，是全世界海拔最高的大型猫科动物，仅分布在中亚12个国家最严酷、最遥远的高山，其种群数量近半分布在中国。它们是高山生态系统的顶级捕食者，是生态系统健康与否的“指示器”。
如今，世界上的雪豹种群正在迅速的消失——过去16年里，野生雪豹的数量已经减少了20%，目前仅有4000-7000只雪豹生活在野外。




—以“拯救”为主题，将充分的设计融入环保，不仅是对雪豹的关注，更是对地球的关注。

为了保护雪山的精灵，同时宣传健康环保的生活理念，膳魔师在今年推出雪豹杯系列产品，希望让随身携带水瓶成为每一个人的习惯，减少一次性杯子的使用，从而减少对自然环境的影响。膳魔师期待以“一滴水”的微小行动，汇聚成“一片海”的环保力量，保护在这个地球上挣扎生存的濒危物种。
雪豹杯沿袭了经典的“一按即开”式设计，人性化的防烫杯口，可以保护娇嫩的嘴唇、避免烫伤的尴尬；




杯身用黑色墨点，寥寥几笔勾勒出半张雪豹头像，而蓝色是它们凝视星空的眼睛；

如果你足够细心，会发现在杯子背面，还有几只透明的脚印，那是雪豹们渐行渐远的足迹……


杯子上所有这些图案的设计都是立体的。除此之外，膳魔师对杯子的包装盒也作了特别的设计，只需要展开就能发现是一副完整的雪豹图。


雪豹杯身上的这些处处充满设计感的小心思，得到了 IF 大奖的充分肯定，让雪豹杯和普通保温杯相比起来，更加时尚潮流。在保留品质的同时又不失低调优雅，也是对环保理念直抒胸臆的表达。





—环保需要更多的人共同努力，每一个看似微小的举动，联合起来也能保护地球。

要知道，每一个随手抛弃的一次性用品，都要花去上数十年，甚至上百年的时间才能在地球上自然降解。这些很难降解的垃圾不但成为自然环境的巨大包袱，也严重威胁到与人们共同在地球上生活的动物们，每年有成千上万的野生动物们因此丧命。
拥有百年历史的膳魔师，一直以来都对生命充满敬意与关护，希望不断通过品牌传递社会责任感，在二战期间和人类的探险历史上，都留下过光辉和荣耀。
也正因为如此，膳魔师希望通过将环保融入生活，来呼吁更多的人关注和保护，不仅仅是雪豹，还有那些一样濒临灭绝或者正在挣扎的物种。请减少使用一次性用品，对这些动物们的栖地环境，做出我们力所能及的小小贡献！
